# 📅 우리학교 학사일정 캘린더

학교 학사일정을 쉽고 빠르게 확인할 수 있는 웹앱입니다.

학교를 검색한 뒤 월별 학사일정을 달력 형태로 확인할 수 있으며, 방학·시험·행사 일정만 골라서 볼 수 있습니다. 모바일에서도 편하게 사용할 수 있도록 제작했습니다.

## 🌐 바로가기

https://sen-vip.github.io/school-calendar/

## 주요 기능

### 🏫 학교 검색

* 학교명 검색
* 시도교육청별 검색
* 선택한 학교 자동 저장
* 새로고침 후에도 선택 학교 유지

### 📅 학사일정 달력

* 월별 학사일정 조회
* 달력 형태 일정 확인
* 날짜별 일정 상세 보기
* 오늘 날짜 강조 표시

### 🔍 일정 찾기

* 방학·개학 필터
* 시험·평가 필터
* 행사·체험 필터
* 일정 키워드 검색

### 🎓 학년별 보기

* 학년별 학사일정 필터
* 전학년 일정 구분 표시
* 학년 대상 일정 확인 가능

### 📤 공유 기능

* 학교 링크 복사
* 이번 달 링크 복사
* 선택 날짜 링크 복사
* 학년 필터 상태까지 포함한 화면 공유
* 공유 링크 접속 시 학교·월·날짜·학년 보기 자동 복원

### 📱 모바일 지원

* 반응형 화면
* 모바일 최적화 UI
* 터치 기반 일정 확인

## 데이터 안내

본 서비스는 학교 학사일정 정보를 기반으로 동작합니다.

학교별 실제 일정은 변경될 수 있으므로 최종 내용은 학교 공지사항을 함께 확인하시기 바랍니다.

## 현재 버전

* GitHub Pages/브라우저 캐시 때문에 기존 app.js/style.css가 남아 보이는 문제를 막기 위해 파일 버전 쿼리 추가
* 달력 렌더링 시 일정 제목 기준으로 최종 색상 분류를 한 번 더 적용
* 토요휴업일/공휴일/대체공휴일 필터·요약·달력칩 색상 일관성 보강

### Beta v3.4.1

* 프록시가 내려주는 기존 일정 type보다 화면 표시명 재판별을 우선 적용
* 토요휴업일을 회색 칩으로 분리
* 공휴일을 앰버 계열로 표시해 시험 일정 빨강과 구분
* 대체공휴일을 연한 앰버/노랑 계열로 별도 표시
* 방학·개학, 시험·평가, 행사·체험 색상 체계 정리
* 일정 종류 필터에 토요휴업·공휴일·대체공휴일 추가

## 업데이트 기록

### 3-4-1 업데이트

* 토요휴업일·공휴일·대체공휴일 색상이 기존 분류값에 밀려 잘못 보이던 문제 수정
* 일정 제목/내용 기준 재판별을 우선 적용해 어린이날·석가탄신일·대체공휴일 색상 분리 보강

### 3-4 업데이트

* 토요휴업일은 회색으로 표시해 반복 휴업 정보를 뒤로 정리
* 공휴일은 앰버, 대체공휴일은 연한 앰버로 표시
* 기말고사 등 시험·평가 빨강과 공휴일 색상이 충돌하지 않도록 분리
* 달력 칩, 일정 목록 배지, 월 요약 배지에 같은 색상 규칙 적용

### 3-3 업데이트

* 공유 기능 고도화
* 학교 링크, 이번 달 링크, 선택 날짜 링크 복사 지원
* 학년 필터를 적용한 상태도 공유 링크에 반영
* 공유 영역을 과한 박스 없이 가볍게 정리
* 초등학교 카드의 좌우 여백과 배지 정렬 보정
