# 우리학교 생활 달력

학사일정, 급식, 시간표를 한 번에 확인할 수 있는 학교생활 캘린더입니다.

학교를 검색해 선택하면 월별 학사일정과 급식, 시간표를 달력에서 확인할 수 있고, 오늘 한눈에 보기와 복사 기능으로 필요한 내용을 바로 공유할 수 있습니다.

## 바로가기

https://sen-vip.github.io/school-life-calendar/

## v1.1 안정판

v1.0의 디자인과 기능 흐름은 유지하고, 내 학교와 학년·반 저장 기능을 정리한 안정판입니다.

## 주요 기능

### 학교 검색

- 전국 학교 검색
- 시도교육청별 검색
- 선택 학교 저장
- 새로고침 후에도 선택 학교 유지

### 학사일정 달력

- 월별 학사일정 조회
- 달력 안 학사일정 배지 표시
- 날짜 클릭 시 상세 학사일정 확인
- 이전 달, 다음 달, 오늘 이동

### 급식정보

- 월별 급식정보 조회
- 달력 안 급식 배지 표시
- 날짜 클릭 시 급식 메뉴 확인
- 오늘 한눈에 보기에서 급식 메뉴 전체 표시
- 급식 칼로리 딱지 표시

### 시간표 조회

- 초등학교, 중학교, 고등학교 시간표 조회 지원
- 학년, 반, 학기 기준 조회
- 조회한 시간표 localStorage 저장
- 달력 안 시간표 배지 표시
- 오늘 한눈에 보기에서 학년·반 딱지 표시

### 오늘 한눈에 보기

- 오늘 학사일정 요약
- 오늘 급식 메뉴와 칼로리 표시
- 저장된 오늘 시간표 표시
- 학교 선택 후 바로 확인 가능한 요약 카드 제공

### 내용 복사

- 오늘 내용 복사
- 선택 날짜 복사
- 메신저, 문자, 카카오톡 등에 붙여넣기 좋은 텍스트 생성
- 복사 성공 안내 표시

### 모바일 지원

- 반응형 화면 구성
- 모바일에서 달력, 카드, 버튼 표시 최적화
- 긴 급식 메뉴 줄바꿈 지원

## 사용 방법

1. 시도교육청과 학교명을 입력해 학교를 검색합니다.
2. 검색 결과에서 학교를 선택합니다.
3. 월별 달력에서 학사일정과 급식 배지를 확인합니다.
4. 날짜를 클릭하면 학사일정, 급식, 시간표 상세를 확인할 수 있습니다.
5. 시간표는 학년, 반, 학기를 입력한 뒤 `시간표 불러오기`를 누르면 조회됩니다.
6. `오늘 내용 복사` 또는 `선택 날짜 복사` 버튼으로 내용을 복사할 수 있습니다.

## Render 프록시 안내

GitHub Pages는 정적 사이트이므로 NEIS API 키를 직접 포함하지 않습니다.

프론트엔드는 Render 프록시 서버를 통해 NEIS API를 호출합니다.

현재 프론트에서 사용하는 프록시 주소:

```text
https://school-life-calendar-proxy.onrender.com
```

프록시 서버에서 사용하는 주요 API 경로:

```text
/api/schools
/api/schedules
/api/meals
/api/timetable
```

Render 환경변수:

```text
NEIS_API_KEY
```
