# 🏫 오늘학교

**학사일정 · 급식 · 시간표를 한눈에**

👉 바로가기: https://sen-vip.github.io/school-life-calendar/

---

## ✨ 무엇을 할 수 있나요?

**오늘학교**는 학교를 검색하면  
우리학교의 **학사일정, 급식, 시간표**를 한 화면에서 확인할 수 있는 웹앱입니다.

매일 아침 가장 궁금한 것들,

- 오늘 학교에 무슨 일정이 있는지
- 오늘 급식은 무엇인지
- 오늘 시간표는 어떻게 되는지

빠르게 확인할 수 있도록 만들었습니다.

---

## 🌷 주요 기능

### 👀 오늘 한눈에

앱에서 가장 먼저 확인하고 싶은 **오늘 정보**를 모아 보여줍니다.

- 오늘 학사일정
- 오늘 급식
- 오늘 시간표
- 오늘 날짜 표시
- 오늘 내용 복사
- 학교·학년·반·월/날짜 공유 링크

학교를 선택하면 자동으로 오늘 날짜가 선택되고,  
**“우리학교 오늘” 제목과 날짜가 먼저 보이도록** 화면 위치를 맞췄습니다.

---

### 📅 학사일정 보기

학교를 선택하면 월별 학사일정을 달력으로 확인할 수 있습니다.

- 학교별 학사일정 조회
- 월 이동
- 오늘 날짜 확인
- 날짜 선택 시 해당 날짜 일정 확인
- 일정이 있는 날은 달력에서 표시

---

### 🍱 급식 보기

선택한 학교의 급식 정보를 확인할 수 있습니다.

- 오늘 급식 확인
- 선택 날짜 급식 확인
- 급식 칼로리 표시
- 알레르기 정보 접기/펼치기

---

### 🕘 시간표 보기

학년과 반을 선택하면 시간표를 확인할 수 있습니다.

- 학년·반별 시간표 조회
- 오늘 시간표 확인
- 선택 날짜 시간표 확인
- 시간표 딱지로 현재 학년·반 표시

---

### 💾 내 학교·학년반 저장

한 번 선택한 학교와 학년·반은 이 기기에 저장됩니다.

다음에 다시 접속하면  
학교를 다시 검색하지 않아도 바로 확인할 수 있습니다.

- 선택 학교 저장
- 학년·반 저장
- 학기 정보 저장
- 저장 초기화 가능

---

### 📋 안내 문구 복사와 공유

오늘 또는 선택한 날짜의 정보를 복사해서  
메신저나 안내문에 활용할 수 있습니다.

- 오늘 안내 복사
- 선택 날짜 안내 복사
- 현재 학교·학년·반·월 공유
- 선택 날짜 공유 링크 생성
- 학사일정·급식·시간표를 보기 좋게 정리

---

## 📱 모바일 사용성

모바일에서도 편하게 볼 수 있도록 화면을 정리했습니다.

- 상단 영역 슬림화
- 오늘 한눈에 카드 우선 노출
- 학교 선택 후 오늘 제목과 날짜가 먼저 보이도록 조정
- 모바일 달력 가독성 개선
- 긴 일정명은 점 표시로 단순화
- 버튼은 모바일에서 누르기 쉽게 정리
- 급식 알레르기 정보는 접기/펼치기로 정리

---

## 🧸 사용 방법

1. 학교명과 지역을 입력해 학교를 검색합니다.
2. 학년·반·학기를 입력합니다.
3. 검색 결과에서 우리학교를 선택합니다.
4. 학교 선택과 동시에 오늘 기준 학사일정·급식·시간표가 적용됩니다.
5. 날짜를 누르면 해당 날짜의 학사일정·급식·시간표가 함께 갱신됩니다.
6. 다른 반 시간표를 보고 싶을 때는 시간표 카드의 `반 바꾸기`를 눌러 바로 변경합니다.
7. 오늘 한눈에에서 일정·급식·시간표를 확인하고 안내 문구를 복사합니다.
8. `공유하기` 또는 `이 날짜 공유`로 현재 상태가 담긴 링크를 보냅니다.

---

## 🛠️ 사용 데이터

본 서비스는 공공데이터 기반 학교 정보를 활용합니다.

- 학교 기본 정보
- 학사일정 정보
- 급식 정보
- 시간표 정보

※ 학교 또는 교육청에서 제공하는 데이터 상황에 따라  
일부 정보가 없거나 늦게 반영될 수 있습니다.

---

## 🔒 저장 방식

선택한 학교와 학년·반 정보는  
사용자의 브라우저에만 저장됩니다.

서버로 전송되지 않으며,  
다른 기기에서는 다시 선택해야 합니다.

저장된 정보는 앱 안의 **저장 초기화** 버튼으로 삭제할 수 있습니다.

학사일정·급식·시간표가 따로 흩어져 있으면 보기 불편합니다.

그래서 우리학교에 필요한 정보를

**오늘학교**는  
학생, 학부모, 선생님 모두가  
오늘의 학교생활을 빠르게 확인할 수 있는 작은 도구입니다.
