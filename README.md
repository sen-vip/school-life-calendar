# 🏫 우리학교 생활 달력

학사일정, 급식, 시간표 정보를 한 화면에서 확인하는 학교생활 캘린더 확장판입니다.

기존 `우리학교 학사일정 캘린더`를 직접 수정하지 않고, 새 레포지토리에서 확장판으로 관리하는 프로젝트입니다.

## 🌐 바로가기

GitHub Pages 배포 후 주소를 입력해 주세요.

## 현재 제공 기능

### 🔍 학교 검색

- 시도교육청 선택
- 학교명 검색
- 검색 결과 카드 표시
- 선택 학교 저장
- 새로고침 후 선택 학교 유지

### 🧭 오늘 한눈에 보기

- 선택 학교 기준 오늘 카드 표시
- 오늘 학사일정 요약
- 오늘 급식 전체 메뉴 요약
- 저장된 오늘 시간표 요약
- 오늘 시간표 학년·반 기준 표시

### 📅 학사일정 달력

- 월별 달력 표시
- 학사일정 API 연결
- 일정이 있는 날짜에 배지 표시
- 날짜 클릭 시 해당 날짜 학사일정 표시
- 이전 달 / 다음 달 / 오늘 이동

### 🍱 급식정보

- 월별 급식정보 API 연결
- 급식 있는 날짜에 배지 표시
- 날짜 클릭 시 메뉴 표시
- 칼로리 정보 표시
- 급식정보 없음 안내

### 🕘 시간표

- 학년·반·학기 입력 UI 제공
- 입력값 자동 저장
- 선택 날짜 기준 시간표 API 조회
- 초등학교/중학교/고등학교 시간표 API 구분
- 교시별 과목 표시
- 조회한 날짜에 달력 안 시간표 배지 표시

## 배포 구조

```text
GitHub Pages = 화면 표시
Render/Vercel = NEIS API 프록시 서버
NEIS Open API = 학교정보 / 학사일정 / 급식정보 / 시간표 데이터
```

GitHub Pages에서는 서버 코드가 실행되지 않으므로, `server` 폴더는 Render 또는 Vercel 같은 별도 서버 환경에 배포해야 합니다.

---

## 프록시 서버 환경변수

```text
NEIS_API_KEY=발급받은_NEIS_API_KEY
```

## API 연결

현재 프론트는 아래 프록시 주소를 사용합니다.

```js
baseUrl: "https://school-life-calendar-proxy.onrender.com"
```

본인 Render 주소가 다르면 `app.js`의 `API_CONFIG.baseUrl` 값을 수정하세요.
